## Quick Example

```typescript
import { LixCache } from 'lix-cache-sdk';

const cache = new LixCache();

// Set a value - types are remembered
await cache.set('user:1', { name: 'Alice', age: 30 });

// Get it back - fully typed automatically!
const user = await cache.get<User>('user:1');
// TypeScript knows: { name: string; age: number } | null
```

## Type-Safe Collections

```typescript
import { z } from 'zod';

// Define schema (generates TypeScript types automatically!)
const UserSchema = z.object({
  name: z.string(),
  age: z.number().min(0).max(150),
  email: z.string().email()
});

// Create validated collection
const users = cache.collection('user:', UserSchema);

// All operations are validated + fully typed!
await users.set('1', {
  name: 'Alice',
  age: 30,
  email: 'alice@example.com'
});

const user = await users.get('1');  // Type: User | null
```

## Cache-Aside Pattern Made Easy

```typescript
// Fetch user from API, cache for 5 minutes
const user = await cache.remember(
  'user:123',
  async () => {
    const res = await fetch('/api/users/123');
    return res.json();
  },
  { ttl: 300 }
);

// ✨ Checks cache first
// ✨ Computes if missing
// ✨ Stores automatically
// ✨ Deduplicates concurrent calls
```

## Performance Features

### Automatic Batching

All operations in the same event loop tick are automatically batched into a single HTTP request.

```typescript
// These 20 operations → Only 1 HTTP request!
await Promise.all([
  cache.set('user:1', { name: 'Alice' }),
  cache.set('user:2', { name: 'Bob' }),
  // ... 18 more operations
]);
```

### Request Deduplication

Multiple simultaneous requests for the same key are automatically deduplicated.

```typescript
// 10 requests for same key → Only 1 operation!
await Promise.all([
  cache.get('user:1'),
  cache.get('user:1'),
  // ... 8 more
]);
```
